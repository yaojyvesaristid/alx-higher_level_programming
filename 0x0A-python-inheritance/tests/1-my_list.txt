Doctest for MyList
==================

Use of printed_sorted
---------------------

import class MyList:
>>> MyList = __import__("1-my_list").MyList

instantiation and manipulation of new object:
>>> my_list = MyList()
>>> my_list.append(5)
>>> my_list.append(2)
>>> my_list.append(0)
>>> my_list.append(-8)
>>> my_list.append(1)
>>> print(my_list)
[5, 2, 0, -8, 1]

basic sorted usage:
>>> my_list.print_sorted()
[-8, 0, 1, 2, 5]

case list has only negative number:
>>> my_list = MyList()
>>> my_list.append(-5)
>>> my_list.append(-2)
>>> my_list.append(-8)
>>> my_list.append(-1)
>>> print(my_list)
[-5, -2, -8, -1]

sorted negative numbers:
>>> my_list.print_sorted()
[-8, -5, -2, -1]

case list is empty:
>>> my_list = MyList()
>>> print(my_list)
[]

sorted empty list::
>>> my_list.print_sorted()
[]
